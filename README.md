# Todo List App

A modern todo list application built with React, Redux, and Tailwind CSS.

## Features

- ✅ Add tasks with title and description
- ✅ Form validation for title and description
- ✅ Smooth loader effect on submit
- ✅ Delete tasks
- ✅ Beautiful, responsive UI

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Redux Toolkit
- Tailwind CSS
- Vite
